# Paper Agent 📑

סוכן חכם לאיתור, משיכה ותצוגה מקדימה של מאמרי מחקר אקדמאיים (לפי שם או DOI), המותאם במיוחד לצרכי מחקר ופיתוח פורנזי בחטיבת מז"פ של משטרת ישראל.

---

## ✨ תכונות עיקריות

* **אימות כניסה מאובטח 🔐:** ממשק התחברות חטיבתי מבוסס סיסמה (מוגדר ב-Secrets של השרת) למניעת גישה לא מורשת.
* **חווית משתמש מתקדמת (UI/UX) 🎨:** עיצוב זכוכית (Glassmorphism) כהה ומודרני, מבוסס פונטים Rubik ו-Assistant, עם תמיכה מלאה בכיווניות לימין (RTL) ליישור לעברית.
* **איתור מאמרים חכם 🔍:** אינטגרציה עם מאגרי Crossref, Semantic Scholar, Unpaywall ושרתי Sci-Hub כדי להשיג את קובץ ה-PDF הטוב ביותר.
* **זיכרון מטמון מובנה (LRU Cache) 💾:** שמירת מאמרים שהורדו בשרת לטעינה מהירה במיוחד בחיפושים עתידיים, עם מנגנון מחיקה אוטומטי למניעת מילוי שטח הדיסק.
* **תצוגה מקדימה (PDF Preview) 📖:** צפייה ישירה בקובץ ה-PDF של המאמר מתוך הדפדפן ללא צורך בהורדה מוקדמת.
* **רישום אסינכרוני (Async Logging) 📊:** תיעוד חיפושים וסטטיסטיקות שימוש ישירות לגיליון Google Sheets באמצעות Apps Script.

---

## 🚀 הוראות הרצה מקומית

1. ודאו שמותקן אצלכם פייתון (גרסה 3.14.4 ומעלה מומלצת).
2. התקינו את הספריות הנדרשות:
   ```bash
   py -m pip install -r requirements.txt
   ```
3. הרריצו את שרת ה-Streamlit:
   ```bash
   py -m streamlit run app.py
   ```
